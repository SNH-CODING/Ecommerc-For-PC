# Ecommerc-For-PC

A simple Ecommerc webiste for computers parts and pre-built PCs, 
from this website you can buy computer parts, peripheral devices 
and pre-built PC.

Tech used :-
- HTML5
- CSS3
- JavaScript
- Bootstrap5
- Django (Backend)
